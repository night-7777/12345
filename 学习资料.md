# 学习资料：HTML 与 Markdown 入门

## 为什么要学 HTML 和 Markdown？

- HTML 是构建网页的基础，了解它可以帮你创建自己的网页内容。
- Markdown 是轻量级文档格式，用于写笔记、博客、文档、README 等。
- 这两种格式结合起来，能让你快速把内容写成结构化、易读的页面。

## 一、HTML 基础

HTML 是一种标记语言，用标签来组织网页内容。

### 1. HTML 基本结构

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>我的网页</title>
</head>
<body>
  <h1>欢迎来到我的网页</h1>
  <p>这是一个简单的 HTML 页面。</p>
</body>
</html>
```

### 2. 常用标签

- `<h1>` 到 `<h6>`：标题
- `<p>`：段落
- `<a href="...">`：链接
- `<img src="..." alt="...">`：图片
- `<ul>`、`<ol>`、`<li>`：列表
- `<strong>`：加粗
- `<em>`：斜体

### 3. HTML 示例

```html
<h2>我的兴趣</h2>
<ul>
  <li>编程</li>
  <li>设计</li>
  <li>学习新技能</li>
</ul>
```

## 二、Markdown 基础

Markdown 是一种用简单标记写文档的方式。

### 1. 常用语法

- 标题：`# 一级标题`，`## 二级标题`
- 段落：直接输入文本
- 换行：行尾加两个空格或空行
- 加粗：`**加粗文本**`
- 斜体：`*斜体文本*`
- 链接：`[文本](https://example.com)`
- 图片：`![替代文本](image.png)`
- 列表：
  - 无序列表：`- 项目`
  - 有序列表：`1. 项目`

### 2. Markdown 示例

```markdown
# 我的学习笔记

这是一个简单的 Markdown 文档。

## 关键点

- 这是一条无序列表
- 这是一条第二项

[点我访问网站](https://www.example.com)
```

## 三、如何练习

1. 在 VS Code 中新建一个 `.html` 文件，写一个简单网页。
2. 在浏览器中打开该文件，查看页面效果。
3. 新建一个 `.md` 文件，写你的笔记或学习日志。
4. 使用 VS Code 的 Markdown 预览功能查看效果。

## 四、推荐资源

- HTML 教程：W3Schools、MDN Web Docs
- Markdown 教程：Markdown 官方语法、GitHub README
- 在线编辑器：CodePen、JSFiddle、StackEdit

## 五、快速实践任务

- 任务 1：写一个包含标题、段落、链接和图片的 HTML 页面。
- 任务 2：写一篇包含标题、列表和链接的 Markdown 笔记。
- 任务 3：把 HTML 页面和 Markdown 文件保存到你的学习文件夹里，方便以后查看。
